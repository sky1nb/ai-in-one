## ‚ú® Features

### üéØ Multi-AI Platform
- **ChatGPT** - OpenAI's powerful conversational AI
- **Claude** - Anthropic's advanced AI assistant
- **Gemini** - Google's multimodal AI
- **Perplexity** - AI-powered search engine
- **NotebookLM** - Research Tool & Thinking Partner 

### ‚ö° Productivity Features
- **Quick Switch** - Switch between AIs with `Ctrl+1-4`
- **Always on Top** - Keep the app visible with `Ctrl+Shift+P`
- **Prompt Library** - Save and reuse prompts with `Ctrl+P`
- **Dark Mode** - Beautiful, eye-friendly interface

### üîê Privacy & Security
- **No Data Collection** - Your conversations stay private
- **Local Session Management** - Secure, persistent logins
- **Isolated Sessions** - Each AI runs in a separate container
- **External Browser OAuth** - Secure Google authentication

## üîß Development

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Setup
```bash
# Clone the repository
git clone https://github.com/sky1nb/ai-in-one.git
cd ai-in-one

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

### Build for Windows Store
```bash
npm run build:store
```
